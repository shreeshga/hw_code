Looking for a job at the Fine Arts Library?

Start by reading through our job descriptions! Note that there may not necessarily be openings at the time you apply.

Circulation Assistant

Stacks Assistant (No openings for Fall 2011)

AV Equipment Assistant


If you are interested in any of the positions, please fill out an application! Both required forms are found below:

Employment Application

Schedule of Availability


You must submit your application materials via email. Make sure to name the files in the following format (replacing the bold text with your name):

Application: LastnameFirstname-Application.doc
Schedule of Availability: LastnameFirstname-Schedule.xls
Resume (optional): LastnameFirstname-Resume.*


In order to expedite the application process, you should also file an I-9 form through the Library Human Resources office in 213 Olin Hall. Note that you do not need to file an I-9 if you have already done so for another job or application.
